## 架構決策備忘(給實作者的背景說明)

這幾個是設計文件沒有寫死數字、實作時必須有個統一解釋的地方,任務內容都已依此展開,照做即可,不用重新決策:

1. **障礙物跨關共通分類**:設計文件每一關的障礙表都用「對應」關聯到其他關的機制(例如都市玻璃雨遮對應叢林斷裂樹枝)。本計畫把這些「對應」直接實作成 4 種共通行為分類,只換外觀與參數:
   - `crumble`(碎裂平台,不扣血):都市玻璃雨遮、叢林斷裂樹枝。
   - `cycleDanger`(安全→警告→危險循環,危險時站上扣血):都市冷氣室外機、叢林食人花。
   - `flyer`(獨立飛行物,不是平台,疊加在平台列間的空中,碰到扣血):都市鴿子群、叢林蜂群。
   - `bounce`(把玩家往上彈,不扣血):都市帆布遮雨棚、叢林彈跳蘑菇。
   - `norm`(安全平台,落地回血邏輯不變)保留自舊版。
2. **危險機率換算**:設計文件的「危險平台總機率」只涵蓋 `crumble` + `cycleDanger`(碎裂平台雖不扣血但會讓你摔,計入「危險」;彈跳與飛行物不計入)。`bounce` 沿用舊版彈簧的固定 7%。`flyer` 是疊加在平台列「之間」的額外物件,不佔平台生成的機率位,用同一條難度曲線乘以 0.5 當作獨立疊加機率(密度不會壓過平台判讀)。
3. **教學段解鎖**:每關前 30 公尺,把該關 4 種障礙依設計文件表格「由上到下」的順序,每 7.5 公尺解鎖一種(第 30 公尺後全部混合),`norm` 一直都在。
4. **關尾小高峰**:每關最後 15 公尺(`depthInLevel > 85`),危險機率直接拉到該關上限,不再用線性內插。
5. **關卡邊界超出處理**:`depth` 超過目前定義的最後一關(本計畫是叢林關結尾 200m)時,難度參數與教學/混合邏輯全部夾在該關的最後一刻(`depthInLevel` 夾在 `[0,100]`),不觸發任何「全破」畫面 —— 全破邏輯要等第五關(`isFinal: true`)在後續計畫加入才會生效,現在只是先把資料結構開好口子,避免之後要動這段邏輯。
6. **金幣造型**:不用圖片素材,程式依主題色盤畫簡單像素圖示(都市 = 圓形金幣、叢林 = 果實),跟愛心的做法一致,不增加裁切工作量。
7. **CITY_MEGA.png 定位修正**:本計畫實作前重新檢視過這個素材包,發現它其實是「上帝視角可愛市民」風格素材包(角色、家具、店面),不是原設計文件想像的「大樓外牆立面」素材。已改用同包內 `SHIPPING_DOCKS_AREA.png` 的天空+雲層+遠景天際線區塊當背景(已精確量到裁切座標,見 Task 6),牆面則用 `CITY_MEGA.png` 裡的磚柱/窗戶小圖示當重複紋理,視覺仍是 GrafxKid 統一色盤,只是不是原本想像的「整片外牆」拼接,風格取捨已足夠合理,不影響美術方向核可範圍(核可的是色盤/氛圍,不是這個實作細節)。

```js
function themeIndexForDepth(depth){
  let idx = 0;
  for (let i = 0; i < THEMES.length; i++){
    if (depth >= THEMES[i].startDepth) idx = i;
  }
  return idx;
}
function currentTheme(){ return THEMES[themeIndexForDepth(depth)]; }
function depthInLevel(depth, theme){
  const span = theme.endDepth - theme.startDepth;
  return Math.max(0, Math.min(span, depth - theme.startDepth));
}
function isTeachingPhase(dInLevel){ return dInLevel < 30; }
function unlockedObstacleCount(theme, dInLevel){
  if (!isTeachingPhase(dInLevel)) return theme.obstacles.length;
  const per = 30 / theme.obstacles.length;
  return Math.max(1, Math.min(theme.obstacles.length, Math.floor(dInLevel / per) + 1));
}
function dangerProbFor(theme, dInLevel){
  if (dInLevel > 85) return theme.dangerProb[1];
  const t = dInLevel / 100;
  return theme.dangerProb[0] + (theme.dangerProb[1] - theme.dangerProb[0]) * t;
}
function pickPlatformCategory(theme, dInLevel, lastWasDanger){
  const unlocked = theme.obstacles.slice(0, unlockedObstacleCount(theme, dInLevel));
  const steppable = unlocked.filter(o => o.category !== 'flyer');
  const dangerKinds = steppable.filter(o => o.category === 'crumble' || o.category === 'cycleDanger');
  const bounceKinds = steppable.filter(o => o.category === 'bounce');
  const totalDanger = dangerKinds.length ? dangerProbFor(theme, dInLevel) : 0;
  const perDanger = dangerKinds.length ? totalDanger / dangerKinds.length : 0;
  const bounceProb = bounceKinds.length ? 0.07 : 0;
  const r = Math.random();
  let acc = 0;
  if (!lastWasDanger){
    for (const o of dangerKinds){
      acc += perDanger;
      if (r < acc) return { category: o.category, obstacleKey: o.key };
    }
  }
  if (r < acc + bounceProb && bounceKinds.length){
    const o = bounceKinds[Math.floor(Math.random() * bounceKinds.length)];
    return { category: o.category, obstacleKey: o.key };
  }
  return { category: 'norm', obstacleKey: null };
}
function shouldSpawnFlyer(theme, dInLevel){
  const unlocked = theme.obstacles.slice(0, unlockedObstacleCount(theme, dInLevel));
  const flyers = unlocked.filter(o => o.category === 'flyer');
  if (!flyers.length) return { spawn: false, obstacleKey: null };
  const prob = dangerProbFor(theme, dInLevel) * 0.5;
  if (Math.random() < prob){
    const o = flyers[Math.floor(Math.random() * flyers.length)];
    return { spawn: true, obstacleKey: o.key };
  }
  return { spawn: false, obstacleKey: null };
}
```

```js
function spawn(y, forceNorm){
  const theme = currentTheme();
  const dInLevel = depthInLevel(depth, theme);
  const w = 50 + Math.random() * 85;
  const x = WALL + Math.random() * (W - 2 * WALL - w);
  let category = 'norm', obstacleKey = null;

    const pick = pickPlatformCategory(theme, dInLevel, lastWasDanger);

  const heartChance = Math.min(.03 + depth * .0005, .14);
  const heart = !forceNorm && category === 'norm' && Math.random() < heartChance;
  const coinChance = Math.min(.10 + depth * .0006, .30);
  const bigCoinChance = 0.02;
  let coin = null;
  if (!forceNorm && !heart && category !== 'flyer' && Math.random() < coinChance){
    coin = Math.random() < bigCoinChance ? 'big' : 'small';

  plats.push({
```

```python
# 下樓梯小遊戲/assets/tools/crop_sprites.py
import json, base64, io, os
from PIL import Image

HERE = os.path.dirname(os.path.abspath(__file__))
ASSETS = os.path.dirname(HERE)

def main():
    with open(os.path.join(HERE, 'manifest.json'), encoding='utf-8') as f:
        manifest = json.load(f)

    lines = ['// 自動產生,勿手動編輯 — 執行 crop_sprites.py 重新產生', 'const SPRITES = {']
    for item in manifest:
        src_path = os.path.join(ASSETS, item['src'])
        im = Image.open(src_path).convert('RGBA')
        box = (item['x'], item['y'], item['x'] + item['w'], item['y'] + item['h'])
        crop = im.crop(box)
        buf = io.BytesIO()
        crop.save(buf, format='PNG')
        b64 = base64.b64encode(buf.getvalue()).decode('ascii')
        lines.append(f"  {item['name']}: 'data:image/png;base64,{b64}',")
        out_preview = os.path.join(HERE, f"preview_{item['name']}.png")
        crop.save(out_preview)
        print(f"  {item['name']}: {item['w']}x{item['h']} <- {item['src']} @({item['x']},{item['y']})  [{len(b64)} base64 chars]  preview: {out_preview}")
    lines.append('};')

    out_path = os.path.join(HERE, 'generated-sprites.js')
    with open(out_path, 'w', encoding='utf-8') as f:
        f.write('\n'.join(lines) + '\n')
    print(f"\n輸出: {out_path}")

if __name__ == '__main__':
    main()
```

```js
let fireflies = null, leaves = null;
function initJungleAmbience(){
  fireflies = [];
  for (let i = 0; i < 5; i++) fireflies.push({ x: Math.random() * W, y: 100 + Math.random() * 400, t: Math.random() * 10 });
  leaves = [];
  for (let i = 0; i < 4; i++) leaves.push({ x: Math.random() * W, y: Math.random() * H, rot: 0, speed: 15 + Math.random() * 10 });
}
function drawJungleAmbience(dt){
  if (!fireflies) initJungleAmbience();
  for (const f of fireflies){
    f.t += dt;
    f.x += Math.sin(f.t * 0.8) * 6 * dt;
    f.y += Math.cos(f.t * 0.6) * 4 * dt;
    const glow = (Math.sin(f.t * 3) + 1) / 2;
    ctx.fillStyle = `rgba(220,255,150,${0.2 + glow * 0.6})`;
    ctx.beginPath(); ctx.arc(f.x, f.y, 2 + glow, 0, 7); ctx.fill();
  }
  for (const l of leaves){
    l.y += l.speed * dt;
    l.rot += dt * 2;
    l.x += Math.sin(l.rot) * 10 * dt;
    if (l.y > H) { l.y = -10; l.x = Math.random() * W; }
    ctx.save();
    ctx.translate(l.x, l.y);
    ctx.rotate(l.rot);
    ctx.fillStyle = Math.floor(l.rot) % 2 === 0 ? '#4a7a3a' : '#6a3a7a';
    ctx.fillRect(-3, -2, 6, 4);
    ctx.restore();
  }
  ctx.fillStyle = 'rgba(20,30,25,0.15)';
  ctx.fillRect(0, H - 60, W, 60);
}
```
