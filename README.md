# 地心大冒險

經典「小朋友下樓梯」玩法的 HTML5 小遊戲，2026-07-16 由 Claude Code 協助郭哲宏建立（原名「像素下樓梯」，2026-07-20 更名）。

## 怎麼玩
- 直接用瀏覽器打開 `index.html` 就能玩（雙擊即可，不需要網路）。
- 也發布了 claude.ai 網頁版，連結見下方「發布資訊」。
- 電腦：← → 方向鍵（或 A / D）移動。
- 手機：按住螢幕左半邊往左、右半邊往右。

## 遊戲規則（設計文件）
- 像素小人從頂部平台出發，畫面持續往上捲動，必須不斷往下踩平台。
- 捲動速度隨深度變快（55 px/s 起，最高 245 px/s）。
- 深度即分數（每 28 px 為 1 公尺），最高紀錄存在瀏覽器 localStorage（key：`downstairs-best`）。
- 血量 10 格：
  - 踩到尖刺平台或碰到頂部尖刺天花板：扣 2 格（受傷後約 1 秒無敵閃爍）。
  - 落地不會自動回血；要撿平台上的愛心才能補血，一顆回 2 格。
  - 愛心只出現在普通／輸送帶平台上，越深出現機率越高（3% 起，最高 14%）。
  - 血量歸零或掉出畫面底部：遊戲結束。
- 四種平台（長短 50–135 px 隨機、垂直間距 72 px）：
  1. 普通磚塊 — 安全，落地回血。
  2. 尖刺 — 扣血（不會連續兩層出現）。
  3. 輸送帶 — 站上去被往左或往右帶（速度 78 px/s）。
  4. 彈簧 — 把角色往上彈。
- 尖刺與輸送帶的出現機率隨深度上升（尖刺最高 22%、輸送帶最高 16%、彈簧固定 7%）。

## 技術說明
- 全部程式都在 `index.html` 一個檔案裡：Canvas 繪圖（360x640 邏輯解析度、像素風縮放）、固定步長 120 Hz 物理更新、WebAudio 合成音效（無外部素材）。
- 角色是程式內定義的像素圖（字元陣列對應顏色），有站立／跑步／落下三種姿勢。
